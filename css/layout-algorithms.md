## contents

### Flow layout

> 流式是文档默认的布局，模仿的是 MS Word 之类的文档布局。

- 块（block）
  - 对应于 Word 中的段落 paragraph
  - 典型 tags 如：div p h1~h6
- Inline
  - 对应于 Word 中的字符
  - 典型 tags 如：a span
  - 默认 align：baseline
  - 另外，image 默认是 inline
- Positioned layout
  - Relative
  - Absolute
  - Fixed
  - Sticky
